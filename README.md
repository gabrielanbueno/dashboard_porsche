# 🏎️ Porsche Sales Dashboard

Dashboard interativo desenvolvido em **HTML, CSS e JavaScript** para análise de vendas da Porsche, com foco em indicadores estratégicos, filtros dinâmicos e visual premium inspirado na identidade visual da Porsche Brasil.

---

## 📌 Objetivo

Disponibilizar uma ferramenta analítica para acompanhar o desempenho das vendas da Porsche, permitindo identificar tendências, modelos mais vendidos, distribuição geográfica e comportamento dos clientes por meio de uma interface moderna e intuitiva.

---

## ✨ Funcionalidades

### 🎯 KPIs

- Total de Vendas
- Receita Total
- Ticket Médio
- Modelo Mais Vendido
- Estado Líder em Vendas

---

### 🔍 Filtros

O dashboard permite filtrar todas as análises por:

- Modelo Porsche
- Ano do Modelo
- Estado
- Cidade
- Forma de Pagamento

Todos os gráficos e indicadores são atualizados automaticamente conforme os filtros selecionados.

---

## 📊 Perguntas de Negócio

O painel responde às seguintes questões estratégicas:

### 🚗 Quais são os principais modelos vendidos por cidade?

Permite identificar quais veículos possuem maior demanda em cada município.

---

### 📅 Qual ano e modelo apresentaram maior volume de vendas?

Ajuda a identificar tendências de mercado e desempenho dos lançamentos.

---

### 📍 Quais estados concentram o maior número de vendas?

Exibe a distribuição geográfica das vendas por estado.

---

### 🏙️ Quais cidades possuem maior participação nas vendas?

Permite identificar os principais mercados consumidores.

---

### 💳 Qual a forma de pagamento mais utilizada?

Mostra a preferência dos clientes entre os métodos de pagamento disponíveis.

---

### 📈 Quais são os carros mais populares em cada cidade?

Gera insights automáticos com base no comportamento das vendas.

---

## 📊 Visualizações

O dashboard apresenta gráficos para:
